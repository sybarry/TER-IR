Remote control via WIFI et tentative d’un remote control via an Android App:

# Update 1 :

Le contrôle du boitier à distance par une connexion WIFI avec le contrôleur en tant que
serveur sur le réseau fonctionne. Une vidéo de démonstration est mise en ligne à l’adresse :
https://www.youtube.com/watch?v=TP9sMozm2yY. Les étapes de réalisation de cette
communication se retrouvent détaillées en forme de tutoriel que l’on a mis en place dans le
document Tuto_WIFI via le chemin : TUTO Getting Started/Connection via WIFI.

# Update 2 :

Création de l’application Android, une application à quatre fonctionnalités :

1/Bouton Connexion : Afin d’établir la connexion entre l’Android et le boitier, 2 /Bouton
Open : afin d’ouvrir les portes à un angle de 90 degrés ,3/Bouton Close : afin de fermer les
portes par un angle de 90 degrés, 4/Bouton Disconnect : pour mettre fin à la connexion
boitier/Android.

Dans le travail à venir, on effectuera les configurations nécessaire au système Android afin
d’établir la connexion entre ce dernier et le boitier en WIFI. Pour ensuite établir une
connexion Bluetooth par la suite.

Le code coté PC pour l’application et le Bluetooth est déjà prêt, il nous reste alors
d’effectuer les configurations nécessaires pour ces deux modes de communication.
