Si vous voulez utiliser la surcouche de communication SurcoucheCom dans un projet, il faut que vous importer le .jar de la surcouche, qui se trouve dans le dossier 
"TER_S2_M1_informatique-2020-2021" ,  dans votre projet en suivan les instructions suivnates :
	- aller dans "Properties" en faissant un clique droit sur le projet
	- aller ensuite dans "Java Build Path"
	- et ensuite dans "Libraries"
	- cliquer sur "Add External JARs"
	- et ajouter le .jar de la surcouche

Si vous modifiez la surcouche de communication, veuillez exporter de nouveau un .jar, remplacez le .jar de la surcouche dans "TER_S2_M1_informatique-2020-2021". Pour ce fait vous
devez :
	- cliqué sur "Export" dans la fenêtre produit par clique droit sur le projet
	- aller dans le dossier "Java" et sélectionner le "AJR file", ensuite appuyez sur "Next"
	- selectionner le dossier "TER_S2_M1_informatique-2020-2021" dans la destination et appuyer sur "Finish"
Et ensuite vous devez remplacer tout les .jar de la surcouche dans les projet qui l'utilisait en réalisant les instructions au début de se document
